The Key Idea

Overview of the CHORUS framework. (1) Calibration phase. Patches are randomly sampled from a calibration set of WSIs and encoded by \(K_v\) pathology foundation models, each producing features in its own native dimensionality. Generalized Canonical Correlation Analysis (GCCA) fits projection matrices \(\{\mathbf{W}_k\}\) that map all encoders into a unified \(D\)-dimensional representation space. (2) Global agreement signal. In the unified space, each feature dimension is characterized by how strongly the projected encoder views correlate. The agreement spectrum \(\boldsymbol{\rho} \in [0,1]^D\) separates dimensions where encoders Do Agree (encoder-invariant tissue structure) from dimensions where they Do Not Agree (encoder-specific variation). (3) Local uncertainty signal. At inference time, each patch is projected by all encoders and the cross-encoder variance is computed. Patches with high variance (red Gaussian) indicate encoder disagreement and are treated as locally uncertain; patches with low variance (green Gaussian) indicate consensus and are treated as informative. CHORUS uses \(\boldsymbol{\rho}\) and \(\boldsymbol{\sigma}^2(x)\) jointly to guide prompt-based MIL aggregation.

Semantic Alignment Calculation: The interpretability of the CHORUS framework is derived from the alignment between visual patch embeddings and clinical token features. For a given image patch \(i\) with visual feature mean \(\mu_i\) and a clinical text prompt tokenized into tokens \(\{t_j\}_{j=1}^N\), the relevance score \(R_{i,j}\) is defined as the normalized dot product in the joint embedding space:

\[ R_{i,j} = \frac{\mu_i \cdot t_j}{\|\mu_i\| \|t_j\|} \]

The final diagnostic heatmap for a specific clinical term \(j\) is generated by aggregating these relevance scores across the slide manifold, providing a spatially-resolved audit of the model's evidence-based reasoning.

Hierarchical Uncertainty Visualization: The visual interpretability maps are generated through a multi-scale alignment process. For each whole slide image (WSI), feature embeddings are extracted across three heterogeneous backbones (UNI, GigaPath, and ResNet50) and projected into a joint latent space via Generalized Canonical Correlation Analysis (GCCA). The diagnostic uncertainty for each image patch \(i\) is quantified by the variance \(\Sigma^2_i\) across the aligned encoder outputs, representing the degree of consensus among the vision models. These patch-level uncertainty scores are then hierarchically mapped onto the original WSI thumbnail using a high-resolution heatmap overlay.

def adaptive_token_selection_chorus(self, features, sigma_sq, rho_v, text_features):
    # Compute patch-level similarity & filter redundant windows
    _, indices = self.compute_patch_similarity_chorus(features, sigma_sq, rho_v, self.window_size)

    # Penalize text relevance by uncertainty (beta * sigma_sq)
    text_relevance = torch.matmul(feats_for_score, text_features.T).mean(-1)
    if self.variant in ("v3_local", "v4_hybrid"):
        patch_penalty = self.beta * sigma_sq.sum(dim=-1)
        text_relevance = text_relevance - patch_penalty

    # Select Top-K tokens based on importance mask
    _, selected_indices = torch.topk(importance_mask, min(self.L_max, N))
    return features[selected_indices], sigma_sq[selected_indices]

def _fuse(self, x_l_list):
    # Align multiple encoders into shared latent space
    z_list = [x.squeeze(0).float() for x in x_l_list]
    
    # mu: latent mean, sigma_sq: per-patch uncertainty
    mu, sigma_sq, _ = self.visual_align(z_list)
    return mu, sigma_sq

# rho: global agreement signal

Stage 3: Spatial Token Compression

def spatial_token_compression_chorus(self, features, sigma_sq, rho_v):
    # Normalize using agreement signal rho
    chunk_norm = rho_weighted_normalize(chunk, rho_v, self.variant)
    
    # Filter adjacent tokens with high similarity in latent space
    sim = F.cosine_similarity(chunk_norm[:-1], chunk_norm[1:], dim=-1)
    keep_mask = sim < self.sim_threshold
    return torch.cat([chunk[:1], chunk[1:][keep_mask]])

Stage 4: Uncertainty-Aware Attention

def cross_attention_chorus(self, queries, keys, values, sigma_sq_v, rho_v, rho_t):
    # Modulate QK by sqrt(rho) to focus on reliable alignment dimensions
    k = k * torch.sqrt(torch.clamp(rho_v, min=0.0))
    q = q * torch.sqrt(torch.clamp(rho_t, min=0.0))

    # Bias attention scores by patch reliability bias
    patch_bias = per_patch_reliability_bias(sigma_sq_v, self.beta)
    scores = (q @ k.transpose(-2, -1)) * scale + patch_bias

Uncertainty Utility Kernels

def reweight_features(mu, sigma_sq, rho, variant, alpha=1.0, tau=0.3):
    # mu_tilde = g(mu, sigma^2, rho)
    if variant == "v5_precision":
        w = rho / (1.0 + alpha * sigma_sq)
        return mu * w
    return mu

def rho_weighted_normalize(x, rho, variant, eps=1e-8, tau=0.3):
    # rho-weighted L2 normalization
    weight = torch.sqrt(torch.clamp(rho, min=0.0))
    xw = x * weight
    return F.normalize(xw, p=2, dim=-1, eps=eps)

def per_patch_reliability_bias(sigma_sq, beta):
    return -beta * sigma_sq.sum(dim=-1)

def modulate_qkv_for_attention(q, k, v, rho_v, rho_t, variant):
    if variant in ("v2_attn", "v4_hybrid"):
        k = k * torch.sqrt(torch.clamp(rho_v, min=0.0))
        q = q * torch.sqrt(torch.clamp(rho_t, min=0.0))
    return q, k, v

GCCA Algorithmic Walkthrough

Step through the regularized MAXVAR-GCCA procedure that aligns heterogeneous pathology foundation models into a shared latent space.

Step 1 / 9
Step 1 — Setup
Multi-Encoder Feature Extraction
Three frozen foundation models encode every patch independently, each producing features in its own native dimensionality \(d_k\).
\[\mathbf{Z}_{s,k} \in \mathbb{R}^{N_s \times d_k}, \quad k \in \{1,2,3\}\]
EncoderDim \(d_k\)
UNI1024
GigaPath1536
ResNet-501024
Step 2 — Centering
Center Each View
Compute the empirical mean of each encoder on a calibration set of ~140K patches and subtract it.
\[\bar{\mathbf{Z}}_k = \mathbf{Z}_k - \mathbf{1}\boldsymbol{\mu}_k^\top\]
The means \(\boldsymbol{\mu}_k\) are saved and reused at inference.
Step 3 — Covariance
Regularized Covariance
Estimate within-view covariance with a ridge term for numerical stability.
\[\tilde{\boldsymbol{\Sigma}}_{kk} = \tfrac{1}{N}\bar{\mathbf{Z}}_k^\top \bar{\mathbf{Z}}_k + \lambda\mathbf{I}, \quad \lambda{=}10^{-3}\]
Step 4 — Whitening
Inverse Square Root
Transform each view to have identity covariance via eigendecomposition.
\[\tilde{\mathbf{Z}}_k = \bar{\mathbf{Z}}_k\;\tilde{\boldsymbol{\Sigma}}_{kk}^{-1/2}\]
After whitening each encoder's features are decorrelated and unit-variance.
Step 5 — Concatenation
Stack Whitened Views
Concatenate all whitened matrices into one wide matrix.
\[\mathbf{M} = [\,\tilde{\mathbf{Z}}_1 \mid \tilde{\mathbf{Z}}_2 \mid \tilde{\mathbf{Z}}_3\,] \in \mathbb{R}^{N \times 3584}\]
Step 6 — SVD
Truncated SVD
Extract the top \(D{=}512\) right singular vectors — the directions of maximum consensus variance.
\[\mathbf{M} = \mathbf{U}\boldsymbol{\Sigma}\mathbf{V}^\top\]
Step 7 — Recovery
Per-Encoder Projections
Partition \(\mathbf{V}\) and undo whitening to recover frozen projection matrices.
\[\mathbf{W}_k = \tilde{\boldsymbol{\Sigma}}_{kk}^{-1/2}\mathbf{V}_k \in \mathbb{R}^{d_k \times 512}\]
Step 8 — Global Signal
Agreement Spectrum \(\boldsymbol{\rho}\)
Measure per-dimension correlation across encoders. Image-independent.
\[\rho_j = \tfrac{1}{3}\sum_{i \lt k}\mathrm{corr}(\mathbf{H}_{i,j}, \mathbf{H}_{k,j})\]
\(\boldsymbol{\rho}\in[0,1]^{512}\) — same for all slides.
Step 9 — Local Signal
Per-Patch Uncertainty \(\boldsymbol{\sigma}^2\)
At inference, compute cross-encoder variance per patch. Image-dependent.
\[\sigma^2(x) = \tfrac{1}{3}\sum_k (\mathbf{h}_k(x) - \boldsymbol{\mu}(x))^2\]
\(\boldsymbol{\sigma}^2 \in \mathbb{R}^{N_s \times 512}\) — different for every slide.
